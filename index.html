Project link

Solve My Maze

An algorithm visualization tool, built using JavaScript and React.js. The user can design a maze, then select an algorithm to solve it, which will then render each step that the algorithm proceeds to take to find the end, visualizing the solution. Implemented an interative Breadth First Search and a Recursive Depth First Search.

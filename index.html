A full stack single page web app inspired by the news reader Feedly, built using JavaScript, React.js, Redux.js, Rails and PostgreSQL. Accessed numerous website's API's to fetch and parse article data and render the data in interactive react components.

Solve My Maze

Project link

Solve My Maze

An algorithm visualization tool, built using JavaScript and React.js. The user can design a maze, then select an algorithm to solve it, which will then render each step that the algorithm proceeds to take to find the end, visualizing the solution. Implemented an interative Breadth First Search and a Recursive Depth First Search.

Project link

StackOverflow Hub

A Chrome Extension allowing you quick access to your StackOverflow account, built using JavaScript, React.js and Redux.js. Implemented an oAuth flow to allow a user to login to their StackOverflow accountfrom the extension. Utilized the Fetch utility to access multiple StackOverflow API's to create an autocomplete search, and access to your inbox for quick checking of messages.
